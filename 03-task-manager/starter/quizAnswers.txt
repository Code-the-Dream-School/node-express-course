1.Quickly explain how middleware is used in an Express application.
  -middleware functions have access to the req and res objects and the next application in the req-res cycle. 
  -the next function passes the response to the next middleware function.
  -middleware essentially executes any code, makes changes to req and res objects, ends the req/res cycle and calls the next middleware in the stack
